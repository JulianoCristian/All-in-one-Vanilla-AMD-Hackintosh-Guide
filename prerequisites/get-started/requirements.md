# 系統要求

## 要求

* **AMD \(FX, Ryzen 或 Zen Athlon\) CPUs**
* **獨立顯示卡（GPU）**
* 如需一個顯示卡列表，請到 [**GPU Buyers Guide**](https://khronokernel-3.gitbook.io/catalina-gpu-buyers-guide/)\*\*\*\*
* 請記得你的**最高支援系統 \(Highest Supported OS\)**
* **由於我還沒有寫完 Legacy BIOS 的方法，你需要有 UEFI 的主機板**

{% hint style="danger" %}
**AMD 的內顯是不支持的！**
{% endhint %}
